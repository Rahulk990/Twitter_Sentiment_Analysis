# Twitter Sentiment Analysis
Performed the task of Sentiment Analysis on the sentiment140 dataset containing 1.6 Million tweets labeled as positive and negative. We have implemented various machine learning models for performing Natural Language Processing on the text of the tweets. 

## _Models Used_
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- Support Vector Classifier
- LSTM Based NN Classifier

## _Py Libraries Used_
- Scikit-Learn
- Keras
- Tensorflow
- NLTK
- Gensim
- Seaborn

## _Results_
The best model achieved an accuracy of 0.747 and an f1 Score of 0.711
